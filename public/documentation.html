Doc page of Jill and Alex's list

This document is a compilation of the movies and shows Jill and Alex have watched or want to watch together

Description URL with query parameters HTTP Method Request Body Format Response Body Format
Gets list of movies /movies GET none JSON object with movie list
Gets details for one movie /movies/:title GET none JSON object with data about a movie
Gets description of a movie by title /movies/:title/description GET none JSON object with description of a movie
Gets year of a movie by title /movies/:title/year GET none JSON object with year of a movie
Gets genre of a movie by title /movies/:title/genre GET none JSON object with genre of a movie
Gets director of a movie by title /movies/:title/director GET none JSON object with director, bio, and birth/death of a movie
Gets info for a director /movies/director/:name GET none JSON object with director info
Gets movies of a certain genre /movies/:genre GET none JSON object with all movies in a genre
Gets featured movies /movies/featured GET none JSON object with all featured movies
Toggles featured or unfeatured /movies/:title/featured PUT none JSON object with message confirming new featured status
Creates a new User /users POST JSON object with info about new user, makes sure username exists in request Message to confirm or deny addition of user
Changes a username /users/:email/:username PUT none Message to confirm or deny update of username by email address
Adds a favorite movie to a user's list /users/:username/favorites/:movie PUT none Array of favorite movies
Deletes a favorite movie to a user's list /users/:username/favorites/:movie DELETE none Message confirming or denying movie deletion
Deletes a user /users/:email DELETE none Message confirming or denying user deletion
