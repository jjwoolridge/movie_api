Doc page of Jill and Alex's list

This document is a compilation of the movies and shows Jill and Alex have watched or want to watch together

Description URL with query parameters HTTP Method Request Body Format Response Body Format
Gets list of movies /movies GET none JSON object with movie list
Gets details for one movie /movies/:Title GET none JSON object with data about a movie
Gets description of a movie by title /movies/:Title/description GET none String object with description of a movie
Gets director of a movie by title /movies/:Title/director GET none JSON object with director, bio, and birth/death of a movie
Gets info for a director /movies/director/:Name GET none JSON object with director info
Gets genre description /movies/genre/:Genre GET none JSON object with description of a certain genre
Gets user information /users/:Username GET none JSON object with message confirming new featured status
Creates a new User /users POST JSON object with info about new user, makes sure username exists in request. Format:
{
"Name" : STRING
"Birthday " : DATE
"Username" : STRING
"Password" : STRING
"Email" : STRING
} 		Message to confirm or deny addition of user
Changes a username /users/:Username POST JSON object with user including new username. Format:
"Username" : STRING
"Password" : STRING
"Email" : STRING
JSON object with user showing updated username.
Adds a favorite movie to a user's list /users/:Username/favorites/:MovieID PUT none JSON object with user and updated favorites list.
Deletes a favorite movie to a user's list /users/:Username/favorites/:MovieID DELETE none Returns JSON object with User stats and movie list withuot movie.
Deletes a user /users/:Username DELETE none Message confirming or denying user deletion
